# BOB Home Care - Professional Website

A modern, bilingual (Arabic/English) website for BOB Home Care, a professional cleaning service specializing in Airbnb properties and home cleaning in Al Alamein and Sidi Abdel Rahman, Egypt.

## Features

- **Bilingual Support**: Full Arabic (RTL) and English (LTR) language support with language toggle
- **Responsive Design**: Mobile-first design that works seamlessly on all devices
- **Modern Animations**: Smooth scrolling, fade-in effects, and interactive elements
- **Services Showcase**: 6 professional cleaning services with detailed descriptions
- **Before/After Gallery**: Real transformation photos showcasing cleaning work
- **Team Section**: Professional team photo display
- **Customer Reviews**: Testimonials and Airbnb reviews section
- **Blog Section**: 6 SEO-optimized blog posts about cleaning topics
- **Contact Form**: Integrated WhatsApp contact form for easy bookings
- **Professional Branding**: BOB Home Care logo and brand colors throughout

## Deployment Options

### Option 1: Vercel (Recommended)
1. Push the project to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "New Project" and import the GitHub repository
4. Click "Deploy"
5. Your site will be live at a Vercel URL

### Option 2: Netlify
1. Push the project to GitHub
2. Go to [netlify.com](https://netlify.com)
3. Click "New site from Git"
4. Select your repository
5. Click "Deploy site"

### Option 3: GitHub Pages
1. Push to GitHub
2. Go to repository Settings → Pages
3. Select "Deploy from a branch"
4. Choose the main branch
5. Your site will be live at `username.github.io/bob-home-care`

## Content Updates

### Adding a New Blog Post
1. Create a new HTML file (e.g., `blog-new-topic.html`)
2. Copy the structure from an existing blog post
3. Update the content, title, and meta information
4. Add a link to the blog post in the blog section of `index.html`

### Updating Images
Replace images in the `images/` folder and update the src attributes in HTML files accordingly.

### Changing Contact Information
Update the WhatsApp number in:
- `index.html` - Contact form section
- `script.js` - WhatsApp integration code
